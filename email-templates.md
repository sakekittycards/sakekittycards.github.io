## Notes

- Always address by first name if you have it.
- Keep it short. Customers read the numbers first, everything else second.
- If a quote is lower than they expected, don't apologize — point to the
  payout tier table and the condition notes. Be direct and kind.
- For graded cards, note the grade and grader in the line item
  (e.g., "Charizard PSA 8 (Base Set Shadowless)").
